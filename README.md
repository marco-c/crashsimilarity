## Running tests

With Python 2:
```sh
# Run all tests
python2 -m unittest discover tests

# Run a specific test
python2 -m unittest tests.test_basic
```

With Python 3:
```sh
# Run all tests
python3 -m unittest discover tests

# Run a specific test
python3 -m unittest tests/test_basic
```
